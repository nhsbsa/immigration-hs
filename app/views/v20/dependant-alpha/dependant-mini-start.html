Apply for a dependant connected to an existing claim

Use this service to:

  • add a dependant to an existing Immigration Health Surcharge (IHS) reimbursement claim
  • create a separate claim for a dependant if their claim period start date is different from the main applicant's

Eligibility

Only use this service if you want to add a dependant to an existing main applicant's claim which has been approved and paid.

The dependant's claim must cover the same period as the main applicant's, or a period within it.

If the main applicant's claim has not been paid yet, you must wait until you receive payment before adding dependants.

Before you start

You'll need:

  • the main applicant's claim reference number
  • the main applicant's date of birth
  • the main applicant's National Insurance number
  • the dependant's personal details
  • the dependant's claim period start date

The service will check that the main applicant claim has been approved and that the dependant is not already included on an existing claim for the same period.
