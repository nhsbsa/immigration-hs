Add or link a dependant to an existing immigration health surcharge reimbursement claim

Use this service to:

  • add a missing dependant to an existing Immigration Health Surcharge (IHS) reimbursement claim
  • create a separate dependant claim if their IHS start date is different from the main applicant's

Eligibility

You can only use this service if:

  • you have already made a claim as the main applicant
  • your main applicant claim has been paid and successful
  • you want to link a dependant to that claim

The dependant's claim must:

  • be linked to a main applicant claim that has already been paid
  • cover the same claim period, or a period within it

If your main applicant claim has not been paid yet, you must wait until you receive payment before adding dependants.

Before you start

You'll need:

  • the main applicant's claim reference number
  • the main applicant's date of birth
  • the main applicant's National Insurance number
  • the dependant's personal details
  • the dependant's claim period start date

The service will check that the main applicant claim has been approved and that the dependant is not already included on an existing claim for the same period.
