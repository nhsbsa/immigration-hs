Dependant application

Apply for a dependant connected to an existing claim

Use this service to:

  • add a dependant to an existing Immigration Health Surcharge (IHS) reimbursement claim
  • create a separate claim for a dependant if their claim period start date is different from the main applicant's

Eligibility

The main applicant's claim must have been approved and paid before using this service to add a dependant to it.

The dependant's 6-month claim must cover the same period as the main applicant's, or have a start date within this period.

Dependant claim period example

Main applicant claim period: 1 January 2022 to 30 June 2022

The dependant's claim period start date must fall within these dates to be added to this claim period.

If the dependant's claim start date is 1 July 2022 or later, it would have to be added to the main applicant's next claim of July to December.

Additional payslips

If the dependant's claim period is the same as the main applicant's, we do not need any additional payslips.

You will need to provide payslips for the main applicant for any dates that do not overlap with the dependant's claim.

Additional payslips example

Main applicant claim period: 1 January 2022 to 30 June 2022

Dependant claim period: 1 March 2022 to 31 August 2022

We would use the March, April, May and June payslips from the main applicant's claim. You would need to provide payslips covering July and August for the dependant's claim.

These payslips must show the main applicant was still working in a health and social care role for these extra months.

Before you start

You'll need:

  • the main applicant's BSA reference number for the claim you are adding your dependant to
  • the main applicant's date of birth
  • the main applicant's National Insurance number
  • the dependant's personal details
  • the dependant's claim period start date
  • any additional payslips for the main applicant that cover the dependant's claim period
  • any extra information that will help us process the application, for example, to cover gaps in employment or name changes

The service will check that the main applicant's claim has been approved and that the dependant is not already included on an existing claim for the same period.
