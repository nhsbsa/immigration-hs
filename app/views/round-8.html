Whats being tested?

This round of research is the first round of research following the sign of the MVP prototype.

Within this round, we will be revisiting and testing the following post MVP features:

  • period end date validation
  • preview file functionality
  • multi file upload on all file upload

Prototype and user journeys

Journey Prototype Description Link
1 Full 2.0 Functionality This prototype includes all post MVP functionalities, and had the period end date validation included within the journey. Prototype link
2 Payslip checker tool This version of the prototype does not have the period end date validation included within the main journey, and instead has a link on the start page to a new standalone payslip checker service where users can choose to enter the dates and get a general verdict on whether they will cover their claim period.

This prototype starts at the new version of the start page (Gov and IHS managed) but will skip to evidence upload on start click.		 Prototype link
3 Preview and multi file upload, but no period end date This is the version of the prototype which entirely scraps period end date validation, if we decide that users do not understand the process.
